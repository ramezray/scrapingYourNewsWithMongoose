# Scraping Your Articles With Mongoose

## Overview:

Scraping Your Articles is a full-stack application that provides  user the ability to select articals and save notes on MongoDB

## Technologies Used:
- Javascript
- MongoDB
- Node.js
- HTML
- CSS
- Handlebars

- NPM Packages Used:
    * express
    * logger 
    * mongoose
    * express-handlebars
    * axios
    * cheerio
